1.1 Introduction to julian

julian returns the number of elapsed days since January 1, 4713 BC. Note that the conventional calendar changed in October 1582 due to the adoption of the Gregorian calendar. The algorithm is based on Peter Duffett-Smith “Astronomy with your personal computer”, 2nd edition. In this algorithm, there is no year 0 B.C. The Macsyma tests is from Jean Meeus “Astronomical Algorithms”, which uses a year 0 B.C. convention. Users should be careful when using years <=0.

1.2 Definitions for julian

Function: julian(day, month, year)

Returns the Julian day number.

Function: modified_julian(day, month, year)

Returns a modified Julian day number that is offset by 2.4 million days.

Function: julian1900(day, month, year)

Returns the number of Julian days since 1900 A.D.

Function: julian1904(day, month, year)

Returns the number of Julian days since 1904 A.D.

Function: iso8601(day, month, year)

The iso8601 standard and the proleptic Gregorian calendar, extends the Gregorian calendar backward from 1582. Julian days start at noon in Greenwich on November 24, 4714 B.C.

Function: calendar(julianDay)

Returns the [day, month, year] given a Julian day number.

Function: calendar8601(julianDay)

Returns the [day, month, year] given a iso8601 day number.

Function: emacs_absolute_day(day, month, year)

Returns an emacs day number given day, month, year.

Function: centuries_j2000(jd)

Returns the number of centuries since 1 January 2000.

Function: sind(x)

Returns sin of the argument given in degrees.

Function: cosd(x)

Returns cos of the argument given in degrees.

Variable: day

The day of the month.

Variable: dec

Declination in degrees.

Variable: E

Equation of time in minutes.

Variable: jd

Julian day number.

Variable: julianDay

Julian day number.

Variable: month

month of year.

Variable: T

(Fractional) number of centuries since 1 January 2000.

Variable: UT

Universal time in hours.

Variable: year

Year number.
